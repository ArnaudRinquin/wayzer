Try me on [Heroku](http://wayzer.herokuapp.com/)

---

### Install

```
git clone git@github.com:ArnaudRinquin/wayzer.git
cd weazer
npm install
```

### Build

```
npm run client:build
```

### Serve

```
npm run server:start
```

### Test

```
npm run test:unit
npm run test:e2e
```

### Dev

In separate shells
```
npm run client:dev
npm run server:dev
npm run test:unit:dev
```
